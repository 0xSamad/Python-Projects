# Miles to KM Converter

A simple and lightweight Graphical User Interface (GUI) application built with Python's `tkinter` module. It allows users to quickly convert distances from Miles to Kilometers.

![Demo](demo/demo.gif)

## How It Works

1. Enter a numerical value representing miles into the input box.
2. Click the **Calculate** button.
3. The program multiplies the miles by `1.609` and displays the rounded result in kilometers directly below.

## Tech Used

- Python
- `tkinter` (standard GUI library)

## Run It Locally

```bash
python main.py
```

No external dependencies are required since `tkinter` comes pre-installed with the standard Python library.
