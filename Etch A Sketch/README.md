# Turtle Sketch App

A simple Python drawing application built using the `turtle` module.

## Features

- `W` → Move forward
- `S` → Move backward
- `A` → Turn left
- `D` → Turn right
- `C` → Clear the drawing and return to the starting position

## Requirements

- Python 3.x
- Built-in `turtle` module

## Run

```bash
python main.py
```

Use the keyboard controls to move the turtle and draw on the screen.
